# 리액트 설치
- 터미널 실행
- node -v (node.js 설치 되었는지 확인 명령어)
- cd Desktop (cd는 이동 명령어)
- mkdir React (React 폴더 생성)
- cd React (생성된 폴더로 이동)
- npx create-react-app my-app(React 폴더에 my-app이라는 리액트가 있는 폴더 생성)
- cd my-app (my-app 폴더 이동)
- code . (visual studio code 실행)
- cmd + ` (vscode 터미널 실행),
- npm start (리액트 실행)
