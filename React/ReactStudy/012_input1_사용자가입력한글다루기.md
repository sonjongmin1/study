# `input`

### input 입력시 코드실행방법

#### Event Handler

- onChange : user가 값을 입력할때마다 실행 (입력이 완료된 후)
- onInput : onChange와 기능 같음, user가 값을 입력할때마다 실행 (입력할때마다 즉시)
- onMouseOver : 마우스를 가져다놯을때 실행
- onScroll : scroll bar를 조작할때 마다 코드 실행<br>
  Event Handler 모르면 찾아보기, 아래 내용은 반드시 숙지
