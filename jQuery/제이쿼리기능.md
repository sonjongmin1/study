# `제이쿼리 이펙트`

### $("선택자").addClass();

- css class 네이밍 추가

### $("선택자").toggleClass();

- css class 네이밍 토글

### $("선택자").show();

- 보이다

### $("선택자").hide();

- 감추다

### $("선택자").toggle();

- 토글 기능

### $("선택자").slideDown();

- 슬라이드 다운

### $("선택자").slideUp();

- 슬라이드 업

### $("선택자").slideToggle();

- 슬라이드 토글

### $("선택자").fadeIn();

- 서서히 보이기

### $("선택자").fadeOut();

- 서서히 감추기

### $("선택자").fadeToggle();

- 서서히 기능 토글

### $("선택자").eq(n);

- Index n번째의 "요소" 뜻

### e.target

- 이벤트 변수 e.target은 객체.속성
  <br>
  <br>

### 주의사항

- 현재의 버전기준에서 화살표함수는 this를 사용할 수 없다.<br>
  function을 쓰면 this 사용가능하다.

### next()

- next()는 다음형제요소

### stop()

- 이펙트이전에 stop()사용하면 누적된 애니메이션은 삭제하고 최후의 애니메이션만 실행한다.

### 활용 예시1

```js
$(document).ready(() => {
  //실행문
  $("선택자").click(() => {
    $("선택자").css("background", "gold");
  });

  $("선택자").click(() => {
    $("선택자").slideToggle(900);
  });

  $("nav a")
    .eq(2)
    .click((e) => {
      $(e.target).css("left", "300px");
    });
});
```

### 활용 예시2

```html
<!DOCTYPE html>
<html lang="en">
  <head>
    <meta charset="UTF-8" />
    <meta name="viewport" content="width=device-width, initial-scale=1.0" />
    <title>Document</title>
    <script src="jquery-3.7.1.min.js"></script>
    <script src="https://code.jquery.com/jquery-3.7.1.js"></script>
    <style>
      #target {
        width: 70px;
        height: 70px;
        background-color: skyblue;
      }
    </style>
  </head>
  <body>
    <input type="button" id="fadeout" value="fadeout" />
    <input type="button" id="fadein" value="fadein" />
    <input type="button" id="hide" value="hide" />
    <input type="button" id="show" value="show" />
    <input type="button" id="slideup" value="slideup" />
    <input type="button" id="slidedown" value="slidedown" />
    <input type="button" id="slideToggle" value="toggle" />
    <input type="button" id="mix" value="mix" />
    <div id="target">target</div>
    <script>
      $(document).ready(function () {
        $('input[type="button"]').click(function (e) {
          let $this = $(e.target);
          // 클릭한 속성의 아이디를 가져오기
          switch ($this.attr("id")) {
            case "fadeout":
              $("#target").fadeOut("slow");
              break;
            case "fadein":
              $("#target").fadeIn();
              break;
            case "hide":
              $("#target").hide();
              break;
            case "show":
              $("#target").show();
              break;
            case "slideup":
              $("#target").slideUp();
              break;
            case "slidedown":
              $("#target").slideDown();
              break;
            case "slideToggle":
              $("#target").slideToggle();
              break;
            case "mix":
              $("#target").fadeOut().fadeIn().delay(1000).slideUp();
              break;
          }
        });
      });
    </script>
  </body>
</html>
```

- 1초 동안 멈춰라 fadeIn().delay(1000)

# `animate`

- 선택자.animate({속성:값},속도,콜백함수)

```html
<!DOCTYPE html>
<html lang="en">
  <head>
    <meta charset="UTF-8" />
    <meta name="viewport" content="width=device-width, initial-scale=1.0" />
    <title>Document</title>
    <script src="jquery-3.7.1.min.js"></script>
    <script src="https://code.jquery.com/jquery-3.7.1.js"></script>
    <style>
      #target {
        position: relative;
        width: 70px;
        height: 70px;
        background-color: skyblue;
      }
    </style>
  </head>
  <body>
    <button id="moveBtn">사용자정의 애니메이션</button>
    <div id="target"></div>
    <script>
      $(document).ready(function () {
        $("#moveBtn").click(function () {
          // 사용자정의 애니메이션
          //   ms 1초 동안에 일어나도록 하겠다.
          $("#target")
            .animate(
              {
                left: "200px",
                opacity: 0.5,
                width: "140px",
                height: "140px",
              },
              1000
            )
            .animate(
              {
                left: "0",
                opacity: 1,
                width: "70px",
                height: "70px",
              },
              1000
            );
        });
      });
    </script>
  </body>
</html>
```

```html
<!DOCTYPE html>
<html lang="en">
  <head>
    <meta charset="UTF-8" />
    <meta name="viewport" content="width=device-width, initial-scale=1.0" />

    <style>
      * {
        margin: 0;
        padding: 0;
      }
      .qDiv {
        padding: 10px 20px;
        border: 1px solid darkgray;
        cursor: pointer;
      }

      .aDiv {
        display: none;
        padding: 10px 30px;
        background-color: lightgray;
      }
    </style>
  </head>
  <body>
    <div class="qDiv">질문. jQuery는 무엇인가요?</div>
    <div class="aDiv">답변. javaScript 라이브러리 입니다.</div>

    <div class="qDiv">질문. jQuery의 장점은?</div>
    <div class="aDiv">답변. 간결한코드 작성입니다.</div>

    <script>
      $(document).ready(function () {
        $(".qDiv").click(function () {
          $(this).next(".aDiv").slideToggle();
        });
      });
    </script>
  </body>
</html>
```
